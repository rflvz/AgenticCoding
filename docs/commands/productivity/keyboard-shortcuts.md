# Atajos de Teclado de Cursor

Guía profesional de atajos de teclado esenciales para trabajar eficientemente con Cursor y maximizar la productividad en el desarrollo.

## Filosofía: Dominar los Atajos

Los atajos de teclado son fundamentales para la productividad. **NO** dependas solo del mouse. Actúa como un **desarrollador profesional** que:

1. **Memoriza los atajos esenciales** para flujos de trabajo comunes
2. **Practica regularmente** para desarrollar memoria muscular
3. **Personaliza atajos** según tus necesidades específicas
4. **Combina atajos** para flujos de trabajo más rápidos
5. **Documenta atajos personalizados** para referencia futura

## Proceso de Uso Profesional de Atajos

### Paso 1: Atajos de Chat (Esenciales)

**⚠️ REGLA CRÍTICA: Estos son los atajos MÁS IMPORTANTES - memorízalos primero:**

#### Abrir Chat

- `Ctrl+L` (Windows/Linux) / `Cmd+L` (macOS) - **Chat rápido** (el más usado)
- `Ctrl+Shift+L` / `Cmd+Shift+L` - Chat en panel lateral
- `Ctrl+I` / `Cmd+I` - Chat inline (para editar código directamente)

#### Cerrar Chat

- `Escape` - Cerrar chat actual
- `Ctrl+W` / `Cmd+W` - Cerrar panel lateral

**Uso profesional:**
- Usa `Ctrl+L` / `Cmd+L` constantemente para acceder rápidamente al chat
- `Ctrl+I` es ideal para ediciones rápidas en el código
- `Escape` te permite cerrar rápidamente y volver al código

### Paso 2: Atajos de Completado de Código

**⚠️ REGLA CRÍTICA: Estos atajos aceleran significativamente la escritura de código:**

#### Aceptar Sugerencias

- `Tab` - **Aceptar sugerencia completa** (el más usado)
- `Ctrl+Right` / `Cmd+Right` - Aceptar palabra por palabra
- `Ctrl+Shift+Right` / `Cmd+Shift+Right` - Aceptar hasta el final de la línea

#### Rechazar Sugerencias

- `Escape` - Rechazar sugerencia actual
- `Ctrl+Shift+Escape` / `Cmd+Shift+Escape` - Rechazar todas las sugerencias

**Uso profesional:**
- `Tab` es tu mejor amigo - úsalo constantemente para aceptar sugerencias útiles
- `Ctrl+Right` te da control fino sobre qué aceptar
- `Escape` te permite rechazar rápidamente sugerencias no deseadas

### Paso 3: Navegación Rápida

**⚠️ REGLA CRÍTICA: La navegación rápida es clave para la productividad:**

#### Comandos Rápidos

- `Ctrl+K` / `Cmd+K` - **Comando rápido** (palette de comandos) - **MUY ÚTIL**
- `Ctrl+P` / `Cmd+P` - Búsqueda rápida de archivos
- `Ctrl+Shift+P` / `Cmd+Shift+P` - Abrir paleta de comandos completa

#### Navegación de Archivos

- `Ctrl+B` / `Cmd+B` - Mostrar/ocultar barra lateral
- `Ctrl+` ` / `Cmd+` ` - Mostrar/ocultar terminal
- `Ctrl+Shift+E` / `Cmd+Shift+E` - Explorador de archivos

**Uso profesional:**
- `Ctrl+K` / `Cmd+K` es esencial para acceder rápidamente a comandos
- `Ctrl+P` / `Cmd+P` te permite abrir archivos instantáneamente
- Combina estos atajos para navegación ultra-rápida

### Paso 4: Edición Avanzada

**⚠️ REGLA CRÍTICA: La edición eficiente multiplica tu productividad:**

#### Selección y Edición

- `Ctrl+D` / `Cmd+D` - Seleccionar siguiente ocurrencia (muy útil para renombrar)
- `Ctrl+Shift+L` / `Cmd+Shift+L` - Seleccionar todas las ocurrencias
- `Alt+Click` / `Option+Click` - Cursor múltiple
- `Ctrl+Alt+Up/Down` / `Cmd+Option+Up/Down` - Cursor arriba/abajo

#### Movimiento

- `Ctrl+Left/Right` / `Cmd+Left/Right` - Mover palabra por palabra
- `Ctrl+Home/End` / `Cmd+Home/End` - Ir al inicio/fin del archivo
- `Ctrl+G` / `Cmd+G` - Ir a línea específica

**Uso profesional:**
- `Ctrl+D` es increíblemente útil para renombrar variables
- Cursor múltiple te permite editar múltiples líneas simultáneamente
- Combina movimiento y selección para ediciones complejas rápidas

### Paso 5: Atajos Específicos de Cursor

**⚠️ REGLA CRÍTICA: Estos atajos aprovechan las capacidades únicas de Cursor:**

#### Comandos de IA

- `Ctrl+L` / `Cmd+L` - Abrir chat con IA
- `Ctrl+K` / `Cmd+K` - Comando rápido con IA
- `Tab` - Aceptar sugerencia de IA

#### Contexto

- `Ctrl+Shift+A` / `Cmd+Shift+A` - Añadir archivo al contexto
- `Ctrl+Shift+C` / `Cmd+Shift+C` - Añadir selección al contexto

**Uso profesional:**
- Añadir archivos al contexto mejora significativamente las respuestas de IA
- Usa selección al contexto para preguntas específicas sobre código

## Flujos de Trabajo Comunes

### Flujo 1: Edición Rápida con IA

```
1. Ctrl+L (abrir chat)
2. Escribir solicitud
3. Tab (aceptar sugerencia)
4. Escape (cerrar chat)
```

### Flujo 2: Renombrar Variable

```
1. Seleccionar variable
2. Ctrl+D (seleccionar siguiente ocurrencia)
3. Repetir hasta seleccionar todas
4. Escribir nuevo nombre
```

### Flujo 3: Navegación Rápida

```
1. Ctrl+P (buscar archivo)
2. Escribir nombre
3. Enter (abrir)
```
